# SSO Demo 单点登录

#### Description
一个简单的单点登录(SSO)的demo.简单来说就是在一个具有多个子系统的系统中，只用登录一个子系统，然后访问其他子系统时不需要再次登录，即“一次登录，多处访问”，能够有效的提升用户体验。
